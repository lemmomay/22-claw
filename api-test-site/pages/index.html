说明
这版很轻，但浏览器天然会更挑环境。
  • 这个 Pages 版由浏览器直接请求目标 API，不经过你的服务端。
  • 如果接口不支持浏览器跨域访问（CORS），这里会失败，但不代表接口本身一定坏了。
  • 页面不会写 localStorage / sessionStorage / IndexedDB；刷新后当前输入和结果都会丢失。
