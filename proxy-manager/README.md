# Proxy Manager

> 🚧 项目规划中，尚未开始开发

一个简单易用的 sing-box/xray 代理管理脚本，针对个人使用习惯定制。

## 📋 项目状态

- **状态：** 规划中
- **优先级：** 中等
- **预计开发时间：** 10-15 天
- **创建时间：** 2026-02-27

## 🎯 设计目标

- 简单易用 - 交互式菜单，一键操作
- 轻量级 - 适合小鸡部署
- 个性化 - 根据使用习惯定制
- 安全可靠 - 自动备份配置

## 📚 文档

详细规划请查看 [PLANNING.md](./PLANNING.md)

## 💡 为什么要做这个项目？

虽然网上有很多成熟的脚本，但：
- 功能太多，不够简洁
- 界面不够友好
- 不符合个人使用习惯
- 想要更多定制化功能

## 🚀 开发计划

### Phase 1: MVP（最小可用版本）
- 基础安装/卸载
- VLESS + Reality 支持
- 简单用户管理
- 交互式菜单

### Phase 2: 功能完善
- 多协议支持
- 流量统计
- 订阅链接
- 监控告警

### Phase 3: 高级功能
- 多用户管理
- 负载均衡
- Web 面板（可选）
